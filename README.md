# snow

#### 介绍
基于springboot的精简的后台权限管理系统。

管理系统功能
已完成
1.用户管理 2.角色管理 3.部门管理 4.菜单管理 5.权限管理 6.登录日志 
待完成
1.字典管理 2.业务日志 3.代码生成

#### 软件架构
软件架构说明
springboot
layui
mybatisplus

#### 安装教程

1. git 导出项目
2. 安装数据库 数据库sql在db文件夹下
2. maven install
3. 运行，SnowApplication.java
